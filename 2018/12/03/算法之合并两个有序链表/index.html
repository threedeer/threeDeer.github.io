算法之合并两个有序链表

题目描述

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

思路

首先要知道什么是https://threedeer.github.io/2018/11/29/%E6%95%B0%E6%8D%AE%E7%BB%93%E6%9E%84%E4%B9%8B%E9%93%BE%E8%A1%A8/和有序链表；例如示例中的 1->2>4,1-3->4；先判断Node1的 1 跟 Node2 的1 比较，相等就把Node1的val作为新的Node的val，Node2作为新的Node的Next；再将较小的那个Node(Node1)的Next跟较大的Node（Node2）比较，依次递归下去，得到的新的Node就是合并之后的.

代码实现

public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
      if (l1 == null) return l2;
      if (l2 == null) return l1;

       ListNode head = null;
       if (l1.val <= l2.val){
           //当l1当前值<=l2的当前值,那就把l1的next取出，将l2再次合并进去递归
           head = l1;
           head.next = mergeTwoLists(l1.next,l2);
       }else {
           //当l1当前值>l2的当前值,那就把l2的next取出，将l1再次合并进去递归
           head = l2;
           head.next = mergeTwoLists(l1,l2.next);
       }

       //最后出来的新的ListNode就是合并之后的数据

       return head;
   }
