Messenger

Messenger是一种轻量级的IPC方案，底层是AIDL；对AIDL进行了封装，一次处理一个请求，所以无需考虑线程同步问题。

步骤

  1. 服务端进程
  • 首先在服务端创建一个Service，来处理客户端的链接请求；
  • 同时创建一个Handler 并通过它来创建一个 Messenger 对象，然后在service的onBind 中返回这个Messenger对象底层的Binder。
  1. 客户端进程
  • 首先绑定服务端Service，绑定成功后用服务端返回的IBinder对象创建一个Messenger；
  • 通过这个Messenger就可以向服务端发送消息，消息类型为Messenger对象
  • 服务端回应客户端就要像服务端一样，创建一个Handler并新建一个Messenger对象，并把这个Messenger对象通过Message的replyTo参数传递给服务端，服务端通过这个参数就能回应客户端。

说了这么多，不如看例子。

实例

服务端

/**
 * @author T
 * @date 2018/12/11
 * 创建一个Service
 */
public class MessengerOfIpcService extends Service {
    private static final String TAG = "MessengerOfIpc";

    /**
     * 创建一个handler
     */
    private static class MessengerHandler extends Handler {
        /**
         * 处理消息
         *
         * @param msg
         */
        @Override
        public void handleMessage(Message msg) {
            switch (msg.what) {
                case 0:
                    Log.i(TAG,"Client：" + msg.getData().getString("msg"));
                    break;
                default:
                    super.handleMessage(msg);
            }
        }
    }

    /**
     * 创建Messenger对象
     */
    Messenger mMessenger = new Messenger(new MessengerHandler());

    @Nullable
    @Override
    public IBinder onBind(Intent intent) {
        //获得底层Binder
        return mMessenger.getBinder();
    }
}

当然 不能忘了在Manifest里面注册我们的服务

<service  
     android:name="com.demo.t.ipc.MessengerOfIpcService"
     android:process=":binderPool" />

客户端

首先当然是创建连接

 /**
 * MessengerOfIpcService
 */

private Messenger mMessenger;
private ServiceConnection messengerConn = new ServiceConnection() {
    @Override
    public void onServiceConnected(ComponentName name, IBinder service) {
        mMessenger = new Messenger(service);
        Message  message = Message.obtain(null,0);
        Bundle bundle = new Bundle();
        bundle.putString("msg","hi！I am Client!");
        message.setData(bundle);
        try {
            mMessenger.send(message);
        } catch (RemoteException e) {
            e.printStackTrace();
        }

    }

    @Override
    public void onServiceDisconnected(ComponentName name) {

    }
};

然后再在onCreate方法里面绑定service

Intent intent = new Intent();
intent.setClass(this,MessengerOfIpcService.class);
bindService(intent,messengerConn, Service.BIND_AUTO_CREATE);

最后别忘了在onDestory中解绑

unbindService(messengerConn);
