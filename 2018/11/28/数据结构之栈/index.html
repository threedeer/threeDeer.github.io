数据结构之栈

栈

栈是限定尽在表头进行插入和删除操作的线性表。

特性

  1. 只能在表头进行插入和删除操作
  2. 先进后出原则存储数据

基本操作

  • 插入：压栈、入栈
  • 删除：出栈、退栈

算法思想

进栈算法(PUSH)

  1. 若 TOP ≥ N 时，则给出溢出信息，作出错处理(进栈前首先检查栈是否已满，满则溢出，不满则作下面处理)；
  2. 置TOP = TOP + 1(栈指针加1，指向进栈地址)；
  3. S(TOP) = X，结束(X为新进栈的元素)；

退栈算法(POP)

  1. 若 TOP ≤ 0，则给出下溢信息，作出错处理(退栈前先检查是否已是空栈，空则下溢出，不空则作下面处理)；
  2. X = S(TOP) （退栈后的元素赋值给X）；
  3. TOP = TOP - 1,结束（栈指针减一，指向栈顶）。

代码实例（实现一个简单的有效括号问题）

public boolean isValidStack(String s) {
    HashMap<Character, Character> map = new HashMap<>();
    map.put(')', '(');
    map.put(']', '[');
    map.put('}', '{');

    Stack<Character> stack = new Stack<>();

    for (int i = 0; i < s.length(); i++) {
        char c = s.charAt(i);

        if (map.containsKey(c)) {
            //右边括号先判断stack是否为空，空的话直接返回false
            //stack不是空再判断与顶层是不是同一类型的括号，不是就返回false

            if (stack.empty() || stack.pop() != map.get(c)) {
                return false;
            }

        } else {
            //左边括号不管，直接入栈
            stack.push(c);
        }

    }

    if (stack.empty()) {
        return true;
    } else {
        return false;
    }

}
