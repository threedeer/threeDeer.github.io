算法之有效括号

题目描述

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true

示例 2:

输入: "()[]{}"
输出: true

示例 3:

输入: "(]"
输出: false

示例 4:

输入: "([)]"
输出: false

示例 5:

输入: "{[]}"
输出: true

思路

主要是通过栈来实现，https://leetcode-cn.com/problems/valid-parentheses/solution/这里有更优的解释和思路描述

代码实现

下面是我自己实现的，看了一下执行效率，不怎么好。暂时还没有看到比较好的方法。

 public boolean isValidStack(String s) {
    if (s.length() % 2 != 0) {
        //奇数  0 的时候返回false
        return false;
    }

    HashMap<Character, Character> map = new HashMap<>();
    map.put(')', '(');
    map.put(']', '[');
    map.put('}', '{');

    Stack<Character> stack = new Stack<>();

    for (int i = 0; i < s.length(); i++) {
        char c = s.charAt(i);

        if (map.containsKey(c)) {
            //右边括号先判断stack是否为空，空的话直接返回false
            //stack不是空再判断与顶层是不是同一类型的括号，不是就返回false

            if (stack.empty() || stack.pop() != map.get(c)) {
                return false;
            }

        } else {
            //左边括号不管，直接入栈
            stack.push(c);
        }

    }

    if (stack.empty()) {
        return true;
    } else {
        return false;
    }

}
