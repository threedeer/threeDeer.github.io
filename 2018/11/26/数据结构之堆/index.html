数据结构之堆

堆

堆（heap）是计算机中的一类特殊的数据结构的统称，通常是一个可以被看做一颗树的数组对象。

特性

  1. 堆中某个节点的值总是 不大于 或者 不小于其父节点的值。
  2. 堆总是一个完全二叉树(叶节点只能出现在最下层和次下层，并且最下面一层的结点都集中在该层最左边的若干位置的二叉树)。
    根节点最大的堆叫做 最大堆或者大根堆；根节点最小的堆叫做最小堆或者小根堆。

基本操作

  • build:建立一个空堆；
  • insert:向堆中插入一个新元素；
  • update：将新元素提升使其符合堆的性质；
  • get：获取当前堆顶元素的值；
  • delete：删除堆顶元素；
  • heapify：使删除堆顶元素的堆再次成为堆。

算法思想

不必将值一个个的插入堆中，通过交换形成堆。
假设根的左右子树都已经是堆结构，并且根的元素名为R，这种情况下有两种可能：

  1. R的值小于或者等于其两个子女，此时堆已经完成；
  2. R的值大于某一个或者全部两个子女的值，此时R应与两个子女中值较小的一个交换，结果得到一个堆，除非R仍然大于其新子女的一个或者全部两个，这种情况继续前述操作，直至到达某一个层使它小于它的子女，或者成为叶节点。

结构图

从这里可以看出，节点数是9个，叶子节点是5个，非叶子节点（父亲节点）是4个；其实多罗列几个就可以得出的一些规律

叶子结点 = (节点数(N) + 1) / 2
父亲节点 = (节点数(N) - 1) / 2

如果存在子节点，那么子节点的下标应该有如下规律:

左子节点 = 2 父节点下标(M)
右子节点 = 2 父节点下标(M) + 1

代码实例（实现最大堆）

/**
 * author       T
 * date         2018/11/26
 * description  实现最大堆
 * 用数组存放完全二叉树中的节点，从上到下，从左到右排序
 * 按序存在数组，子节点的值小于等于父节点的值
 **/
public class Heap {
    private int[] data;
    private int capacity;   //当前节点数组
    private int count; //容量

    public Heap(int mCapacity) {
        data = new int[capacity + 1]; //索引0不存在节点，所以长度需要加一
        capacity = mCapacity;
        count = 0;
    }

    /**
     * 将一个无序的数组构造成一个最大堆
     * 相当于堆排序
     *
     * @param arr 无序数组
     * @param n   节点数
     */
    public Heap(int[] arr, int n) {
        data = new int[n + 1];
        capacity = n;
        for (int i = 0; i < n; i++) {
            data[i + 1] = arr[i];
        }
        count = n;

        for (int i = count / 2; i >= 1; i--) {
            //i = count / 2;i是最后一个叶子节点的父节点（最后一个非叶子节点）
            down(i);
        }
    }

    /**
     * 将小的节点下沉
     * @param i
     */
    private void down(int i) {
        while (2 * i <= count) {  //有左子节点
            int j = 2 * i;// 这轮循环，data[i]和data[j]交换位置

            if ((j + 1) <= count && (data[j + 1] > data[j])){//有右子节点，且右子节点更大
                j += 1;
            }
            if (data[i] >= data[j]){//父节点大于等于子节点，退出循环
                break;
            }
            swap(data,i,j);
            i = j; //当值交换后，i的值被应该为当前位置，即下次循环的时候初始化。

        }
    }

    /**
     * 交换数组中两个值
     * @param arr 数组
     * @param a 位置a
     * @param b 位置b
     */
    private void swap(int[] arr,int a,int b){
        int c = arr[a];
        arr[a] = arr[b];
        arr[b] = c;

    }

    /**
     * 将小的节点上浮
     * @param i
     */
    private void up(int i){
        while (i > 1 && (data[i/2] < data[i])){
            //data[i/2] 当前节点的父节点，
            //不是根节点，且当前父节点小于当前节点  交换
            swap(data,i,i/2);
            i = i/2; //更新索引
        }
    }

}

复杂度

n个节点的堆，高度d = log2^n.根为第0层，则第i层的节点个数2i，考虑下一个元素在读中乡下移动的距离，大约一半的节点深度为d-1，不移动（叶）。四分之一的节点深度为d-2，最多乡下移动一层，节点数为前一层的一半，而子树高度加一。

时间复杂度 O(n);

由于堆有logn层深，插入节点，删除普通元素和删除最小元素的平均事件代价和时间复杂度都是

O(logn).
