算法之最长公共前缀

问题描述

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 “”。

示例 1:

输入: ["flower","flow","flight"]
输出: "fl"

示例 2:

输入: ["dog","racecar","car"]
输出: ""
解释: 输入不存在公共前缀。

说明:
所有输入只包含小写字母 a-z 。

思考

字符串数组中公共前缀，那么首先考虑特殊情况，空数组，里面空字符串这很明显就应该直接返回空字符串；数组长度为1的时候，直接返回里面唯一的字符串即可；
普通情况下，如果里面有相同的，我想到的是先拿第一字符串出来，经过双重循环来判断。第一次循环字符串，从第一个字符开始，每次拿这个字符与数组后面的字符串通过startsWith()方法进行比较，如果后面的字符串是以当前截取的字符串开始，那么继续循环，否则跳出循环；
自己这种实现方法虽然简单直接，但是总觉得不优雅，后面再网上找了个二分法实现，记录一下。

代码实现

代码一

public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0) {
        //特殊情况
        return "";
    }
    if (strs.length == 1) {
        //特殊情况
        return strs[0];
    }
     String result = "";
    for (int j = 1; j <= strs[0].length(); j++) {
        boolean noExist = false;
        String val1 = strs[0].substring(0, j);
        for (int k = 1; k < strs.length; k++) {
            if (strs[k].startsWith(val1)) {
                continue;
            } else {
                noExist = true;
                break;
            }

        }
        if (noExist) {
            break;
        } else {
            result = val1;
            continue;
        }
    }
    return result;
}

二分法实现

解析

首先需要找出最短的字符串，最长的前缀长度不会超过此字符串，然后使用二分法查找方式，这里主要涉及到二分法查找，如果不清楚可以先了解一下。

//二分法查找
public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0)
        //特殊情况
        return "";
    int minLen = Integer.MAX_VALUE;
    for (String str : strs)
        //找出最短字符串的长度
        minLen = Math.min(minLen, str.length());
    int low = 1;
    int high = minLen;
    while (low <= high) {
    //利用二分法查找实现
        int middle = (low + high) / 2;
        if (isCommonPrefix(strs, middle))
            low = middle + 1;
        else
            high = middle - 1;
    }
    return strs[0].substring(0, (low + high) / 2);
}

private boolean isCommonPrefix(String[] strs, int len) {
    String str1 = strs[0].substring(0, len);
    for (int i = 1; i < strs.length; i++)
        if (!strs[i].startsWith(str1))
            return false;
    return true;
}
