算法之罗马数字转整数

描述：

罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。

字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000

例如， 罗马数字 2 写做 II ，即为两个并列的 1。12 写做 XII ，即为 X + II 。 27 写做 XXVII, 即为 XX + V + II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 IIII，而是 IV。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 IX。这个特殊的规则只适用于以下六种情况：

I 可以放在 V (5) 和 X (10) 的左边，来表示 4 和 9。
X 可以放在 L (50) 和 C (100) 的左边，来表示 40 和 90。
C 可以放在 D (500) 和 M (1000) 的左边，来表示 400 和 900。
给定一个罗马数字，将其转换成整数。输入确保在 1 到 3999 的范围内。

示例 1:

输入: "III"
输出: 3

示例 2:

输入: "IV"
输出: 4

示例 3:

输入: "IX"
输出: 9

示例 4:

输入: "LVIII"
输出: 58
解释: L = 50, V= 5, III = 3

示例 5:

输入: "MCMXCIV"
输出: 1994
解释: M = 1000, CM = 900, XC = 90, IV = 4

思考

  1. 根据字符串去做比较，每个字符代表不同的意思，利用for循环遇到什么加上即可，但是其中有6种特殊情况需要处理：IV、IX、XL、XC、CD、CM；很显然，一个数据中不可能同一个情况出现两次，只需要特殊处理特殊情况就行。所以循环遍历字符串，将罗马数字转换成对应数字，再将特殊情况处理并减去即可；需要注意的是，特殊处理的时候第一次加上了某个数，再减去的话需要减去双倍的；比如IV，执行的时候是 1+5 = 6，而实际结果应该是4。
  2. 当我们观察罗马字符串，大多时候从右向左数值应该是递增的，比如：LVIII，VI等，但是一旦出现左边数值比右边大，那么就说明出现上述的特殊情况，如 IX,IV,MCMXCIV等等，这个时候只要将右边的值减去左边的值就可以得到当前位置的确切值了。

具体实现

方法一：

 public int romanToInt(String s) {
    int result = 0;
    int length = 0;

    while (length < s.length()) {
        switch (String.valueOf(s.charAt(length))) {
            // I X C 可能因为右边的内容而有不同的表现值
            case "I":
                result += 1;
                break;
            case "X":
                result += 10;
                break;
            case "C":
                result += 100;
                break;
            case "V":
                result += 5;
                break;
            case "L":
                result += 50;
                break;
            case "D":
                result += 500;
                break;
            case "M":
                result += 1000;
                break;
            default:
                break;
        }
        length++;
    }

    if (s.contains("IV") || s.contains("IX")) {
        result -= 2;
    }
    if (s.contains("XC") || s.contains("XL")) {
        result -= 20;

    }
    if (s.contains("CD") || s.contains("CM")) {
        result -= 200;
    }


    return result;
}

public int romanToInt2(String s) {
    if (s == null || s.length() == 0) return -1;

    HashMap<Character, Integer> map = new HashMap<Character, Integer>();
    map.put('I', 1);
    map.put('V', 5);
    map.put('X', 10);
    map.put('L', 50);
    map.put('C', 100);
    map.put('D', 500);
    map.put('M', 1000);
    int len = s.length(), result = map.get(s.charAt(len - 1));
    for (int i = len - 2; i >= 0; i--) {
        if (map.get(s.charAt(i)) >= map.get(s.charAt(i + 1)))
            result += map.get(s.charAt(i));
        else
            result -= map.get(s.charAt(i));
    }
    return result;
}
