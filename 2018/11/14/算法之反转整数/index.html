算法之反转整数

2018-11-14
#算法
算法

反转整数

题目描述

给定一个 32 位有符号整数，将整数中的数字进行反转。

示例 1:
输入: 123
输出: 321

示例 2:
输入: -123
输出: -321

示例 3:
输入: 120
输出: 21

注意:
假设我们的环境只能存储 32 位有符号整数，其数值范围是 [(−2)^31, 2^31 − 1] 。根据这个假设，如果反转后的整数溢出，则返回 0。

思路

这里要将数字进行反转，也就是将低位变换成高位，高位变换成低位；既然是整数，最容易想到的办法自然是通过运算实现；所以不难想到通过取余取商来的方法。每次除以10，取余取商之后，再将余数保留，往后每次再乘以10累加余数，这样低位就能依靠每次乘以10变成高位，高位通过每次除以10变成余数之后累加起来成为低位；具体代码实现如下。

$2^{log_2{n}}$

代码实现

 public int reverse(int x) {
    int outNum = 0; //输出数字
    int m = 0; // 余数
    while (Math.abs(x) > 0) {
        if (Math.abs(outNum)  > (Integer.MAX_VALUE - Math.abs(x % 10)) / 10) {
            //判断是否溢出
            return 0;
        }
        m = x % 10;
        x = x / 10;
        outNum = m + outNum * 10;
    }
    return outNum;
}

步骤解析

首先溢出问题。
应该在计算之间就考虑是否溢出；

if(m + outNum * 10 > Integer.MAX_VALUE)

当结果大于整型的最大值的时候，溢出（负数形式不能忘记），当然这也写肯定不行，因为这种写法永远返回false，应该适当变换一下

if (Math.abs(outNum)  > (Integer.MAX_VALUE - Math.abs(x % 10)) / 10)

然后就是取余，这里将余数取出来，从个位开始；
再后面是取商，这里将保留去掉余数之后的商值，作为下次循环的源数据；
最后就是计算结果值，将要输出的数乘以10提高一位，将计算的余数加上，进入下一次循环；
当商为0的时候循环结束，输出结果值。
